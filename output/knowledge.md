## Mandarin

**爱**
- pinyin : ài
- sens : aimer
- composants : 爫 (griffe/main), 冖 (couvrir), 𠂇 (main gauche), 又 (main droite)
- mnemonique : tenir quelqu'un par la main
- traditionnel : 愛 (avec 心, cœur)

**八**
- pinyin : bā
- sens : huit
- note : sens originel : "diviser", représente la séparation, 8/6 est le plus divisible

**做**
- pinyin : zuò
- sens : faire, agir, travailler
- composants : 人 (homme), 口 (bouche), 攵 (main qui tient un bâton)

**坐**
- pinyin : zuò
- sens : s'asseoir
- composants : 从 (deux hommes), 土 (assis par terre)
